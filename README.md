# Mizuki Plugin Meme

Mizuki Plugin Meme 是开发组内部维护的表情包管理插件，用于整理本地图库、图片上传、删除、溯源、文件夹锁定和缓存管理等能力。

## 项目定位

该插件面向 Mizuki Bot 的群聊图片互动场景，主要用于管理表情包素材和统一图片输出体验。

核心能力包括：

- 本地表情包图库管理。
- 图片上传、删除、随机抽取和来源记录。
- 文件夹锁定或分类管理。
- 常见图片格式处理，包括 jpg、png、gif、webp。
- 图片缓存整理。

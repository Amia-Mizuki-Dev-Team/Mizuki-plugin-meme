# Nonebot-plugin-maimeme

✨ 随机发送maimaiDX表情包插件 ✨

## 功能特性
- 支持OneBot/QQ协议
- 自动缓存管理
- 多格式支持（jpg/png/gif/webp）

## 安装
使用nb-cli安装：
```bash
nb plugin install nonebot-plugin-maimaimeme
```

## 配置
```dotenv
# .env.prod
maimai_meme_path="data/maimai/meme"  # 表情包存储路径
```

## 使用方式
发送指令：
```
mai
```
